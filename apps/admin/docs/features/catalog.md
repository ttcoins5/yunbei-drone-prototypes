# 商品管理
## 路由与代码位置
- 路由：`categories`、`products`、`product-edit`
- 代码：`js/modules/catalog.js`
## 当前功能
- 分类增改删、图标选择、启停和排序。
- 商品列表、新建/编辑、轮播图、多规格、业务属性、富文本介绍和评价展示选择。
## 数据结构
- `state.categories[]`、`state.products[]`、`productReviews[]`。
- 编辑草稿、删除指针、评价草稿和富文本状态均在 `state`。
## 操作与业务规则
- 分类或商品存在关联数据时禁止删除。
- 商品图片最多 9 张，第一张为封面。
- 评价勾选随“保存商品”生效；创建商品时生成 ID 和编号。
## 与其他模块的依赖
- 依赖 `core/files.js`、`core/rich-editor.js`、公共分页、弹窗和行操作。
