# 飞手入驻
## 路由与代码位置
- 路由：`pilot-applications`、`pilot-review`
- 代码：`js/modules/pilot-onboarding.js`
## 当前功能
- 入驻申请列表、个人/公司资料、资质设备展示、通过和驳回审核。
## 数据结构
- `pilotApplications[]`；`state.viewingPilotAppId`。
## 操作与业务规则
- 仅待审核记录显示审核动作；已审核记录只读。
- 公司主体额外展示公司资料和水印执照。
## 与其他模块的依赖
- Dashboard 只读使用待审核申请数据。
- `pilots.js` 复用公司和资质展示 helper。
## 已知占位行为
- 审核只修改内存状态，驳回原因不持久化。

- 2026-06-15：申请、共享飞手资料 helper 和审核动作迁移到独立模块。
