# 轮播图配置

## 路由与代码位置
- 路由：`carousel`
- 代码：`js/modules/carousel.js`
## 当前功能
- 在「小程序配置」父级菜单下作为「轮播图配置」子页面独立展示。
- 管理 1 个视频和最多 5 张图片，支持上传、启停、排序、删除。
## 数据结构
- `state.media[]`：`id`、`type`、`name`、`url`、`poster`、`enabled`。
## 操作与业务规则
- 视频最多 1 个并固定首位；图片最多 5 张。
- 图片可上移、下移，视频不可移动。
## 与其他模块的依赖
- 依赖 `core/files.js`、公共行操作、Toast 和 render。
## 已知占位行为
- 只保存文件名，不上传或持久化文件。
