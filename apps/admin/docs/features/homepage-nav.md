# 导航配置

## 路由与代码位置
- 路由：`homepage-nav`
- 代码：`js/modules/homepage-nav.js`

## 当前功能
- 在「首页配置」父级菜单下作为「导航配置」子页面独立展示。
- 页面改为纯后台列表，不再展示手机可视化预览。
- 固定 8 个首页入口：序号 1-4 为大图片入口，序号 5-8 为小图片入口。
- 使用蓝绿色简约科技风样稿裁切出的 8 张图片作为测试数据。

## 数据结构
- `state.homepageNav.items[]`：`id`、`size`、`title`、`image`、`imageName`、`jumpType`、`link`、`enabled`。
- `state.homepageNavImageUrls[]`：本地临时替换图片的 Object URL。

## 操作与业务规则
- 列表固定展示 8 行，不支持新增、删除入口。
- 入口类型由序号固定决定，1-4 为大图片入口，5-8 为小图片入口。
- 图片已自带入口标题文案，后台不再配置标题和副标题。
- 跳转类型仅支持内链和外链；内链填写 `/` 开头的小程序路径，外链填写 `http://` 或 `https://` 开头的链接。
- 可选择图片替换单个入口展示图。
- 可启用或停用入口。
- 大图片入口和小图片入口分别支持组内排序，不跨类型移动。
- 保存只进行原型校验和反馈；刷新页面后恢复初始 mock 数据。

## 与其他模块的依赖
- 依赖 `core/files.js`、公共弹窗、按钮、Toast、render 和右侧页面说明栏。

## 已知占位行为
- 本地图片使用浏览器 Object URL，仅在当前页面会话内有效。
- 当前导航固定为 8 个入口，不支持调整数量。
