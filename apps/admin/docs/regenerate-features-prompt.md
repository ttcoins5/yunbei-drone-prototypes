# 模块功能文档维护提示

完整功能已拆到 `docs/features/`，后续不再扫描所有 JavaScript 并覆盖生成一份超长清单。

## 单模块更新

```text
请先读取 FEATURES.md 中的全局规则与模块索引，再读取：
- docs/features/{模块文档}.md
- js/modules/{模块文件}.js
- 模块文档明确列出的必要依赖

根据当前代码更新该模块文档的“当前功能、数据结构、操作与业务规则、依赖、占位行为、本轮变更、验收结果”。
不要读取或修改其他无关模块。完成后在 FEATURES.md 的“最近变更”追加一条精简记录。
```

## 全局架构变更

只有修改 `js/core`、模块注册协议、全局路由或跨模块规则时，才更新 `FEATURES.md` 的公共架构和跨模块规则。
